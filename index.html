Tutorial: Google Replication

Before We Start the Tutorial

We will build a replica of the google.com homepage during this tutorial. The techniques you’ll learn in the tutorial are fundamental to building any website, and mastering it will give you a deep understanding of HTML & CSS.

The tutorial is divided into several sections:

  • Setup for the Tutorial will give you a starting point to follow the tutorial.
  • Overview will teach you the fundamentals of HTML elements & CSS styles.
  • Completing the Website will teach you the most common techniques in Web development.

You don’t have to complete all of the sections at once to get the value out of this tutorial. Try to get as far as you can — even if it’s one or two sections.

What Are We Building

In this tutorial, we’ll show how to build the Google website with search bar and all.

You can see what we’ll be building here: Final Result. If the code doesn’t make sense to you, or if you are unfamiliar with the code’s syntax, don’t worry! The goal of this tutorial is to help you understand React and its syntax.

We recommend that you check out the Google replica before continuing with the tutorial. There are three sections we will be building: Navigation, Search Bar & Footer.

You can close the Google replica once you’re familiar with it. We’ll be starting from a simpler template in this tutorial. Our next step is to set you up so that you can start building the website.

Overview

Now that you’re set up, let’s get an overview of HTML and CSS!

What Is HTML?

HTML is a syntax we use to display words and images on a website. Don't worry if you aren't 100% familiar with all of this, you will learn as you go along.

  • Use angle brackets to make HTML elements. <p> & </p>
  • Add content inside the opening and closing tags to display words. <p>Paragraph Text Here</p>
  • Elements require both an opening and closing tag, otherwise it wont get displayed properly, as above.
  • However, two elements that are in this tutorial require only the opening tag. A.K.A. a void tag <img> & <br> Images will require an additional attribute to display an image and the br is just a line break.
  • I can't stress enough the importance of writing out each opening and closing tags correctly. Here is an example of elements opening and closing tags we will use in the tutorial
    • <a> </a>
    • <br>
    • <button> </button>
    • <img>
    • <li> </li>
    • <span> </span>
    • <ul> </ul>

What Is CSS?

We use CSS to style our websites using Selectors (choosing HTML elements to style) and then assign Property:Value; pairs to give style to the selected HTML. Don't worry if you aren't 100% familiar with this, you will learn as you go along.

  • Whereas HTML has tags, CSS has selectors. Selectors are the names given to styles in internal and external style sheets.
  • There are many CSS selectors we can use to style HTML, we will focus on two types in this tutorial: Element Selector and Child Selector
  • For each selector there are “properties” inside curly brackets, which simply take the form of words such as color, font-weight or background-color.
  • A value is given to the property following a colon (NOT an “equals” sign). Semi-colons are used to separate the properties. e.g. body{ color:tomato; }

Website Head

In the <head></head> of an HTML document we include tags so the broswer can use them to improve user experience. Link a title or and icon for a page, or to include a .css file so we can add style to the site.

Below are the necessary updates you need to make to the index.html file.

  • on the first link, set href attribute to "favicon.ico"
  • add content to the website title element
  • on the last link, set href attribute to link to "main.css"
Your index.html file should look like this when you're done.
          <!DOCTYPE html>
<html>
  <head>
    <!-- WEBSITE ICON -->
    <link rel="shortcut icon" href="favicon.ico">
    <!-- WEBSITE TITLE -->
    <title>Google</title>
    <!-- FONT AWESOME ICON CSS - check out fontawesome.com -->
    <script src="https://use.fontawesome.com/349da334fb.js"></script>
    <!-- NORMALIZE CSS -->
    <link rel="stylesheet" href="css/normalize.css">
    <!-- CUSTOM CSS -->
    <link rel="stylesheet" href="css/main.css">
  </head>
  <body>

  </body>
</html>

Website Navigation

Finally, it's time to code! We are going to start inside the <body> tag in index.html and add the necessary code to recreate the Google homepage navigation.

Navigation HTML

We are going to create a box to contain our nav using a div element, followed by a unordered list with some list items and anchors inside each list item.

We strongly recommend typing code by hand as you’re working through the tutorial and not using copy/paste. This will help you develop muscle memory and a stronger understanding.

Below are the necessary updates you need to make to the index.html file.

  • create a div element with a class attibute set to "navigation"
  • inside that div, create a ul element
  • inside that ul, create 4 li elements each with its own a element nested inside
  • on the first anchor element add an href attribute set to "https://mail.google.com/mail/?tab=wm" and the content between the tags to "Gmail"
  • on the second anchor element add an href attribute set to "https://www.google.com/imghp?hl=en" and the content between the tags to "Images"
  • on the third anchor element add an href attribute set to "https://about.google/intl/en/" and the class attribute to "fa fa-th"
  • on the fourth anchor element add an href attribute set to "https://accounts.google.com/SignOutOptions?hl=en"
  • inside the fourth anchor element add an img element, set the class attribute to "profile" and the src attribute to "img/chris.jpg"
    • now add a picture of yourself to the img folder, inside the project folder. update the img src to match your file name. Your picture should be showing up instead of mine.
Your index.html file should look like this when you're done.
          <!DOCTYPE html>
<html>
  <head>...</head>

  <body>
    <div class="navigation">
      <ul>
        <li>
          <a href="https://mail.google.com/mail/?tab=wm">Gmail</a>
        </li>
        <li>
          <a href="https://www.google.com/imghp?hl=en">Images</a>
        </li>
        <li>
          <a href="https://about.google/intl/en/" class="fa fa-th"></a>
        </li>
        <li>
          <a href="https://accounts.google.com/SignOutOptions?hl=en">
            <img class="profile" src="img/chris.jpg">
          </a>
        </li>
      </ul>
    </div>
  </body>
</html>

Navigation CSS

We are going to add some styles to our navigation as well as some site styles (this allows a site to look the same on most browsers)

We strongly recommend typing code by hand as you’re working through the tutorial and not using copy/paste. This will help you develop muscle memory and a stronger understanding.

Below are the necessary updates you need to make to the main.css file. We will start with sitewide styles

  1. select all elements (*)
    • add a color of #000
  2. select the body element (body)
    • add a color of #222
    • add a margin of 0
  3. select all the anchor elements (a)
    • add a color of #222
    • add a text-decoration of none
  4. select all the anchor pseduo hover elements (a:hover) (what happens when a user hovers over a link)
    • add a color of #222
    • add a text-decoration of underline
Your main.css file should look like this when you're done. 
          /** SITE STYLES **/
* {
  color: #000;
}
body {
  color: #222;
  margin: 0;
}
a {
  color: #222;
  text-decoration: none;
}
a:hover {
  color: #222;
  text-decoration: underline;
}

/** SITE NAVIGATION **/

With those sitewide styles out of the way we can move on to the navigation styles. Below are the necessary updates you need to add to the main.css file.

  1. select all div elements with a class of navigation (div.navigation{})
    • add a height of 75px
  2. select all ul elements inside any div elements with a class of navigation (div.navigation ul{})
    • add a float of right
    • add a margin-right of 25px
    • add a margin-top of 0
  3. select all li elements inside any div elements with a class of navigation (div.navigation li{})
    • add a display of inline-block
    • add a font-size of 13px
    • add a margin of 5px 5px 0 0
    • add a padding of 5px 10px 0 0
    • add a vertical-align of bottom
  4. select all a elements inside any div elements with a class of navigation (div.navigation a{})
    • add a color of #626065
    • add a font-size of 14px
  5. select all the anchor pseduo hover elements inside any div elements with a class of navigation (div.navigation a:hover{})
    • add a color of #000
  6. select all elements with a class of fa (.fa{})
    • add a cursor of default
    • add a font-size of 16px !important
    • add a opacity of 0.7
    • add a vertical-align of top
  7. select all the pseduo hover elements with a class of fa (.fa:hover{})
    • add a cursor of pointer
    • add a opacity of 1
  8. select all elements with a class of profile (.profile{})
    • add a border-radius of 25px
    • add a height of 32px
    • add a position of relative
    • add a width of 32px

  margin-right: 25px;
  margin-top: 0;
}

  font-size: 13px;

Website Search Bar

Now we move on to the middle of the website, this includes the search bar and buttons.

Search Bar HTML

We are going to create a box to wrap this search just as we did with thte navigation, followed two boxes to contain the image, and buttons separately and last an input for our search bar.

Below are the necessary updates you need to make to the index.html file.

  • create a div element with a class attribute set to "cta"
  • inside that div, create another div element with a class attibute set to "logo", an input element and another div element with a class attibute set to "search"
  • inside the div with class="logo", create an img element, set the src attribute to "img/google.png" and the alt attribute to "Google Logo"
  • on the input element, set the type attribute to "text" and the placeholder attribute to "Search Google or type a URL" and add the autofocus attrbute
  • inside the div with class="search", create two button elements,
  • add the text content of "Google Search" to the first button and the text content of "I'm Feeling Lucky"" to the second button
Your index.html should look like this when you're done.
          <!DOCTYPE html>
<html>
  <head>...</head>

  <body>
    <div class="navigation">...</div>

    <div class="cta">
      <div class="logo">
        <img src="img/google.png" alt="google logo">
      </div>
      <input type="text" placeholder="Search Google or type a URL" autofocus>
      <div class="search">
        <button>Google Search</button>
        <button>I'm Feeling Lucky</button>
      </div>
    </div>
  </body>
</html>

Website Footer

On to the last part of the website, the bottom footer links!

Footer HTML

We are going to craete one last box with two unordered lists inside. Like our navigation before we will add list item elements with anchor elements inside of each and then add content and links.

Below are the necessary updates you need to make to the index.html file.

  • create a div element with a class attibute set to "footer"
  • inside that div, create two ul elements - the first with a class attibute set to "left" - the second with a class attibute set to "right"
  • inside each ul, create three li elements each with its own a element nested inside
  • on the first anchor element, inside the left ul, add an href attribute set to "https://www.google.com/intl/en/ads/?fg=1" and the content between the tags to "Advertising"
  • on the second anchor element, inside the left ul, add an href attribute set to "https://www.google.com/services/?fg=1" and the content between the tags to "Business"
  • on the third anchor element, inside the left ul, add an href attribute set to "https://www.google.com/intl/en/about.html?fg=1" and the class attribute to "About"
  • on the first anchor element, inside the right ul, add an href attribute set to "https://www.google.com/intl/en/policies/privacy/?fg=1" and the content between the tags to "Privacy"
  • on the second anchor element, inside the right ul, add an href attribute set to "https://www.google.com/intl/en/policies/terms/?fg=1" and the content between the tags to "Terms"
  • on the third anchor element, inside the right ul, add an href attribute set to "https://www.google.com/preferences?hl=en" and the content between the tags to "Settings"
